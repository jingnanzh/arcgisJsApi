Create a 3D map:

      3D scene: Create a SceneView and set the container property to viewDiv. Set the SceneView position by creating a camera and setting the x, y, z coordinates to -118.80800, 33.96100, 2000 and the tilt to 65.
      Learn more about the camera settings in the documentation.
      https://developers.arcgis.com/javascript/latest/guide/create-a-starter-app/
